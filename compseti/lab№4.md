# Лабораторная работа №4
## Диагностика работоспособности персональных компьютеров

**Цель работы:** Проводить диагностику компьютера с помощью программ-диагностик.

**Оборудование:** ПК, виртуальная машина Windows 7, программа PC Wizard, Средство диагностики DirectX.

---

### Краткие теоретические сведения

Программы для диагностики компьютера позволяют проверить конфигурацию компьютера (количество памяти, ее использование, типы дисков и т.д.), а также проверить работоспособность устройств компьютера (прежде всего жестких дисков). Они позволяют выявить «намечающиеся» дефекты дисков (возникающие из-за износа магнитной поверхности диска) и предотвратить потерю данных, хранящихся на диске.

С помощью тестового программного продукта или диагностического комплекса можно определить ряд несложных неисправностей аппаратуры или неправильного конфигурирования системных файлов. Диагностика может выполняться как для системы в целом, так и для отдельных модулей:

- системной платы
- памяти (стандартной, дополнительной и расширенной)
- видеоподсистемы
- жестких дисков
- приводов флоппи-дисков
- клавиатуры
- портов (последовательных и параллельных)
- координатных устройств
- приводов компакт-дисков (CD-ROM)
- устройств, имеющих SCSI-интерфейс

#### Программа PC Wizard

PC Wizard — программа, которая предоставляет обширную информацию обо всех установленных на компьютере компонентах: память, материнская плата, устройства для хранения и записи информации, видеоподсистема, сетевые устройства, модемы, принтеры и т.д., включая разнообразные данные об операционной системе — версию системы, установленные шрифты, библиотеки, WinSock, активные процессы, имеющиеся модули и сервисы и т.п. [citation:2][citation:8]

Кроме этого, PC Wizard позволяет протестировать быстродействие процессора, жестких дисков и CD/DVD-привода. Поддерживаются 32- и 64-битные ОС. [citation:1][citation:3]

### Часть 2. Диагностика с помощью средства диагностики DirectX

#### Краткие теоретические сведения

**Средство диагностики DirectX** — это системный инструмент, предназначенный для тестирования, диагностики и изменения рабочего состояния аппаратного обеспечения. Данная программа отображает сведения о компонентах и драйверах интерфейса Microsoft DirectX. Программа позволяет проверить работу аудио- и видеокарты, а также установить связь с мультимедийными службами. С помощью средства диагностики также можно отключить некоторые средства аппаратного ускорения. [citation:4][citation:6]

#### Порядок выполнения работы

1. Чтобы запустить Средство диагностики DirectX, нажмите комбинацию клавиш **Win+R** и введите в поле запроса команду `dxdiag`. [citation:9]
2. При запуске диагностического средства DirectX выполняется автоматическое тестирование всех компонентов интерфейса. С отчетами проверок можно ознакомиться в поле **Примечания**, расположенном внизу каждой вкладки окна Средство диагностики DirectX.
3. Начиная с первой вкладки **Система**, последовательно щелкайте на кнопке **Следующая страница** внизу окна, чтобы перейти от одной вкладки к другой и проверить, все ли в порядке. В поле **Примечания** каждого окна должна быть желанная фраза **«Неполадок не найдено»**. [citation:10]

### Контрольные вопросы (устно)

#### 1. Причины возникновения проблем с совместимостью ПО

| № | Причина | Описание |
| :--- | :--- | :--- |
| 1 | **Разрядность операционной системы** | 32-битные программы могут работать на 64-битной ОС, но не наоборот. 64-битные программы не запускаются на 32-битной Windows. |
| 2 | **Версия операционной системы** | Программа, разработанная для Windows 10, может не работать на Windows 7 из-за отсутствия необходимых API-функций и библиотек. [citation:3] |
| 3 | **Отсутствие необходимых компонентов** | Многие программы требуют установленных Microsoft Visual C++ Redistributable, .NET Framework или DirectX определенной версии. |
| 4 | **Устаревшие или неподписанные драйверы** | Средство диагностики DirectX может указать на неподписанный драйвер, что может вызывать проблемы с совместимостью и стабильностью системы. [citation:6] |
| 5 | **Аппаратные требования** | Программа может требовать определенный объем ОЗУ, видеопамяти или поддержку特定ных инструкций процессора (SSE2, AVX). |
| 6 | **Конфликт версий DirectX** | Игра может требовать DirectX 11, а видеокарта поддерживает только DirectX 9. [citation:10] |
| 7 | **Антивирусное программное обеспечение** | Некоторые антивирусы могут блокировать работу легитимных программ, ошибочно принимая их за вредоносное ПО. |
| 8 | **Региональные стандарты** | Программа может быть несовместима с форматом даты, времени или десятичного разделителя, принятым в определенной стране. |

#### 2. Методы уменьшения проблем с совместимостью ПО

| № | Метод | Описание |
| :--- | :--- | :--- |
| 1 | **Использование режима совместимости Windows** | В свойствах исполняемого файла (.exe) можно выбрать запуск программы в режиме совместимости с более ранней версией Windows (например, Windows 7 или Windows XP). |
| 2 | **Установка необходимых компонентов** | Установка Microsoft Visual C++ Redistributable (всех версий), .NET Framework (последней версии) и DirectX Runtime. |
| 3 | **Запуск от имени администратора** | Некоторые программы требуют повышенных привилегий для корректной работы. Для этого в свойствах файла нужно выбрать «Запуск от имени администратора». |
| 4 | **Обновление драйверов** | Установка последних версий драйверов для видеокарты, звуковой карты и других устройств с официального сайта производителя. [citation:6] |
| 5 | **Использование виртуальных машин** | Запуск устаревших программ, несовместимых с современными ОС, в виртуальной машине (например, Windows 7 на Windows 10/11, как в данной лабораторной работе). [citation:3] |
| 6 | **Отключение антивируса** | Временное отключение антивирусного ПО при установке или запуске программы, если оно ошибочно блокирует работу. |
| 7 | **Изменение региональных стандартов** | Настройка формата даты, времени и десятичного разделителя в соответствии с требованиями программы. |
| 8 | **Использование портативных версий** | Применение Portable-версий программ, которые не требуют установки и не изменяют системный реестр, что снижает риск конфликтов. [citation:3] |
