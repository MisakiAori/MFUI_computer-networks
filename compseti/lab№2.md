# Лабораторная работа №2
## Оценка качества работы жесткого диска

**Цель работы:** Оценить качество работы жесткого диска с помощью специальных программ (CrystalDiskMark и CrystalDiskInfo).

**Оборудование:** ПК, программы CrystalDiskMark, CrystalDiskInfo.

---

### Краткие теоретические сведения

- **CrystalDiskMark** — приложение для измерения скорости последовательного и случайного чтения/записи накопителя. Позволяет задавать размер блоков данных, количество проходов и объем тестового файла.
- **CrystalDiskInfo** — утилита для анализа состояния диска на основе технологии S.M.A.R.T. (Self-Monitoring, Analysis and Reporting Technology). Показывает температуру, количество ошибок, переназначенных секторов, время наработки и общее техническое состояние устройства.

---

### Ход выполнения работы

#### Часть 1. Тестирование производительности (CrystalDiskMark)

1. Установите программу **CrystalDiskMark**.
2. Запустите данную программу.
3. Установите параметры тестирования:
   - **Количество запусков тестов:** `6`
   - **Объем тестируемых данных:** `1000 MiB`
   - **Раздел жесткого диска:** `C:`
4. Нажмите кнопку **All** для запуска всех тестов одновременно.

### Контрольные вопросы (устно)

#### 1. Что такое жесткий диск?

**Жесткий диск (HDD — Hard Disk Drive)** — энергонезависимое запоминающее устройство, в котором данные записываются на вращающиеся магнитные пластины с помощью движущихся считывающих/записывающих головок. Предназначен для долговременного хранения операционной системы, программ и пользовательских данных.

#### 2. Внутреннее устройство жесткого диска

Типовая структура жесткого диска включает следующие компоненты:

- **Гермоблок** — герметичная зона с магнитными пластинами и головками.
- **Магнитные пластины** — одна или несколько круглых пластин из алюминия или стекла, покрытые ферромагнитным слоем.
- **Считывающие/записывающие головки** — устройства на подвижных рычагах для чтения и записи данных.
- **Катушка и постоянный магнит** — система управления перемещением головок.
- **Шпиндельный двигатель** — вращает пластины с постоянной скоростью.
- **Печатная плата (контроллер)** — содержит процессор, кэш-память, микросхему управления двигателем и интерфейс SATA.

#### 3. Причины возникновения проблем с совместимостью

| № | Причина | Пример |
| :--- | :--- | :--- |
| 1 | Разные типы разъемов | Подключение SATA диска к IDE контроллеру без переходника |
| 2 | Ограничения размера диска в BIOS | Старая материнская плата не видит диск больше 2 ТБ |
| 3 | Неподдерживаемая файловая система | Диск в формате APFS не читается в Windows |
| 4 | Разрядность операционной системы | 32-битная Windows не загружается с GPT диска |
| 5 | Устаревшие драйверы контроллера | Windows не устанавливается на NVMe диск без драйвера |
| 6 | Конфликт интерфейсов на материнской плате | M.2 SSD отключает два SATA порта |

#### 4. Методы уменьшения проблем с совместимостью

| № | Метод | Описание |
| :--- | :--- | :--- |
| 1 | Обновление BIOS/UEFI | Установка последней версии прошивки материнской платы |
| 2 | Использование переходников | Применение адаптеров SATA-USB, M.2-SATA |
| 3 | Установка правильных драйверов | Загрузка драйверов с сайта производителя контроллера |
| 4 | Выбор файловой системы | Использование exFAT или NTFS для совместимости с разными ОС |
| 5 | Правильная инициализация диска | Выбор GPT для современных систем, MBR для старых |
| 6 | Проверка спецификаций | Изучение руководства материнской платы перед покупкой накопителя |
