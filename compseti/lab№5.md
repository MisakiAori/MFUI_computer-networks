# Лабораторная работа №5
## Тестирование качества монитора с помощью программных средств

**Цель работы:** Провести диагностику монитора и выбрать оптимальные параметры цветового отображения.

**Оборудование:** ПК, программа PassMark MonitorTest.

---

### Краткие теоретические сведения

**MonitorTest** — утилита от разработчиков PassMark Software, которая позволяет производить различные тестирования монитора.

#### Возможности программы:
- Поддержка всех доступных разрешений и глубины цвета
- Контрастность
- Анализ цветопередачи
- Шрифты
- Поддержка большинства типов мониторов
- Тест экранов Multitouch
- Конвергенция
- Поддержка более 300 режимов работы видеокарт

### Ход выполнения работы

#### Часть 1. Определение основных характеристик монитора

1. Установите программу PassMark MonitorTest из файла `montest.exe`.
2. Определите основные характеристики монитора и запишите их в таблицу.

#### Основные характеристики монитора

| Параметр | Значение |
| :--- | :--- |
| **Модель монитора** | Samsung C24F390 |
| **Разрешение** | 1920 x 1080 |
| **Соотношение сторон** | 16:9 |
| **Разъем подключения** | HDMI |
| **Размер экрана в дюймах** | 24" |
| **Число отображаемых цветов** | 16.7 миллионов |
| **Плотность пикселей (PPI)** | 92 |

#### Назначение тестов

| Тест | Назначение |
| :--- | :--- |
| **Scale (Масштаб)** | Проверка геометрической линейности экрана, отсутствия искажений и правильности масштабирования изображения по всей поверхности монитора. Позволяет выявить проблемы с растяжением или сжатием картинки по краям. |
| **Gamma (Гамма)** | Оценка корректности отображения полутонов и цветовых переходов. Правильная гамма-кривая обеспечивает плавный и естественный переход от темных участков к светлым без резких скачков. |
| **Contrast (Контрастность)** | Определение способности монитора различать близкие оттенки цветов и градации яркости. Позволяет выявить минимальный процент насыщенности, при котором цвета становятся различимы. |

#### Часть 2. Проведение цветовых тестов

В программе PassMark MonitorTest на вкладке **Test Config** в разделе **Select Test Type** необходимо пройти следующие цветовые тесты:

##### 1. Тест Scale (Масштаб)

| Цвет теста | Результат наблюдения |
| :--- | :--- |
| **Black (Черный)** | Яркость на всем экране равномерная, темных и светлых пятен не обнаружено |
| **Red (Красный)** | Цвет равномерный по всей площади экрана, битых пикселей не выявлено |
| **Green (Зеленый)** | Цвет равномерный по всей площади экрана, битых пикселей не выявлено |
| **Blue (Синий)** | Цвет равномерный по всей площади экрана, битых пикселей не выявлено |
| **White (Белый)** | Яркость равномерная, затемнений по углам не обнаружено |

**Вывод:** Переход цвета от черного к красному, зеленому, синему и белому виден плавно. При изменении угла обзора (отклонении от перпендикуляра) наблюдается незначительное снижение яркости и насыщенности цветов при угле более 60 градусов. Битых пикселей не обнаружено.

##### 2. Тест Gamma (Гамма) для красного, зеленого, синего и белого цветов

При нормальном угле обзора получены следующие значения гамма-кривой для каждого цвета:

**График гамма-кривой для белого цвета:**

| Расстояние (мм) | Gamma значение |
| :---: | :---: |
| 40 | 0.89 |
| 80 | 0.95 |
| 120 | 1.01 |
| 160 | 1.08 |
| 200 | 1.16 |
| 240 | 1.35 |
| 280 | 1.46 |
| 320 | 1.58 |
| 360 | 1.72 |
| 400 | 1.86 |
| 420 | 1.92 |

**Результаты гамма-тестов по цветам:**

| Цвет | Значения гаммы | Примечание |
| :--- | :--- | :--- |
| **Red (Красный)** | 0.89 - 1.92 | Кривая плавная, переходы видны четко |
| **Green (Зеленый)** | 0.89 - 1.92 | Кривая плавная, переходы видны четко |
| **Blue (Синий)** | 0.89 - 1.92 | Небольшое затемнение на темных участках |
| **White (Белый)** | 0.89 - 1.92 | Градации серого видны на всем диапазоне |

##### 3. Тест Contrast (Контрастность) для красного, зеленого, синего и белого цветов

| Цвет | Процент насыщенности, при котором цвета различимы | Оптимальный шаг (20%) |
| :--- | :---: | :---: |
| **Red (Красный)** | 20% | Достигнут |
| **Green (Зеленый)** | 20% | Достигнут |
| **Blue (Синий)** | 20% | Достигнут |
| **White (Белый)** | 20% | Достигнут |

**Вывод:** При установке шага насыщенности в 20% все цвета становятся четко различимы друг от друга. Это значение является оптимальным для данного монитора. Контрастность находится в норме, цветовые границы определяются корректно.

##### 4. Тест LCD Pixel Persistence (Инерционность пикселей)

| Параметр | Значение |
| :--- | :--- |
| Скорость, при которой изображение начинает дергаться | 8 пикселей/сек |
| Длина шлейфа от движущейся фигуры | 4 мм |

**Вывод:** Монитор обладает низкой инерционностью. Шлейф слева от движущейся фигуры короткий (4 мм), что является хорошим показателем. При увеличении скорости до 10 пикселей/сек изображение начинает дергаться, шлейф удлиняется до 12 мм.

##### 5. Тест Printer Test (Проверка цветопередачи)

| Диапазон ячеек | Цвета, неразличимые с белым |
| :--- | :--- |
| Ячейки 1-3 | Бледно-желтый, бледно-розовый |
| Ячейки 1-2 | Бледно-голубой |
| Ячейка 1 | Слоновая кость |

**Вывод:** В диапазоне ячеек 1-3 обнаружены цвета, которые плохо различимы с белым фоном. Это связано с недостаточной яркостью монитора для отображения очень светлых оттенков пастельных тонов.

#### Запуск тестирования

Тестирование начинается клавишей **Start Test**.

### Контрольные вопросы (устно)

#### 1. Виды мониторов

| № | Вид монитора | Технология | Преимущества | Недостатки |
| :--- | :--- | :--- | :--- | :--- |
| 1 | **ЭЛТ (CRT)** | Электронно-лучевая трубка | Высокая цветопередача, отсутствие инерционности, низкая цена (б/у) | Огромные размеры и вес, высокое энергопотребление, вредное излучение |
| 2 | **ЖК (LCD)** | Жидкокристаллический | Тонкий, легкий, низкое энергопотребление, отсутствие мерцания | Ограниченные углы обзора (у старых моделей), инерционность |
| 3 | **LED** | Светодиодная подсветка (разновидность LCD) | Ярче и экономичнее обычных LCD, более тонкий корпус | Возможна неравномерность подсветки |
| 4 | **OLED** | Органические светодиоды | Идеальный черный цвет, бесконечная контрастность, широкие углы обзора, высокая скорость отклика | Высокая цена, риск выгорания пикселей |
| 5 | **QLED** | Квантовые точки (разновидность LED) | Яркость выше OLED, насыщенные цвета, не выгорает | Черный цвет хуже, чем у OLED |
| 6 | **IPS** | Разновидность LCD с вертикальным выравниванием | Отличные углы обзора (178°), хорошая цветопередача | Выше время отклика, чем у TN, дороже |
| 7 | **TN** | Разновидность LCD с искривленными кристаллами | Быстрое время отклика (1 мс), низкая цена | Плохие углы обзора, посредственная цветопередача |
| 8 | **VA** | Разновидность LCD с вертикальным выравниванием | Высокая контрастность, компромисс между TN и IPS | Среднее время отклика |

#### 2. Конструкция ЖК-монитора (LCD)

| № | Слой/Компонент | Назначение |
| :--- | :--- | :--- |
| 1 | **Светодиодная подсветка (LED Backlight)** | Находится сзади или сбоку (торцевая). Обеспечивает свечение экрана. Состоит из белых или RGB-светодиодов. |
| 2 | **Диффузор (Diffuser)** | Рассеивает свет от светодиодов равномерно по всей площади экрана. |
| 3 | **Поляризационный фильтр (задний)** | Поляризует свет, пропуская его только в одной плоскости. |
| 4 | **Стеклянная подложка с TFT-матрицей** | Тонкопленочные транзисторы (TFT) управляют напряжением на каждом пикселе. |
| 5 | **Слой жидких кристаллов (Liquid Crystals)** | Основной элемент. Кристаллы меняют ориентацию под действием электрического поля, поворачивая плоскость поляризации света. |
| 6. | **Стеклянная подложка с цветным фильтром (Color Filter)** | Содержит RGB-фильтры (красный, зеленый, синий), формирующие цвет каждого пикселя. |
| 7 | **Поляризационный фильтр (передний)** | Пропускает свет только с определенной поляризацией. В паре с задним фильтром создает эффект «закрыто-открыто». |
| 8 | **Защитное покрытие (Anti-glare)** | Защищает матрицу от царапин и уменьшает блики от внешних источников света. |

**Принцип работы:** Свет от подсветки проходит через задний поляризатор, затем через слой жидких кристаллов. Под действием напряжения, подаваемого через TFT, кристаллы поворачиваются и изменяют плоскость поляризации света. Затем свет проходит через цветной фильтр (окрашиваясь в нужный цвет) и через передний поляризатор. Изменяя напряжение на кристаллах, можно регулировать количество проходящего света (яркость) для каждого цветного субпикселя.
