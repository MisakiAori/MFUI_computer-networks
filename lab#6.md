# Лабораторная работа №6
## Определение основных характеристик и тестирование видеосистемы ПК

**Тема программы:** Видеосистема персонального компьютера.

**Цель работы:** Изучить современные видеокарты на графических процессорах NVIDIA, AMD (ATI) и технологии объединения видеокарт.

**Время выполнения:** 1 час

**Оборудование:** Учебный персональный компьютер.

**Программное обеспечение:** Операционная система, презентация, тестовые программы (GPU-Z, FurMark, FluidMark).

---

### Теоретические основы

Для выполнения данной лабораторной работы выбраны 2 утилиты: **GPU-Z v0.6.0** и **FurMark v1.10.2**. GPU-Z выдает лишь информацию о видеокарте, а FurMark проводит тестирование.

#### GPU-Z

GPU-Z — это программа для вывода информации о графическом адаптере, которая поддерживает и карты NVIDIA, и ATI/AMD[citation:2][citation:8]. Она поможет узнать:

- модель видеокарты
- интерфейс подключения
- какой используется графический процессор (версия BIOS, номер ревизии чипа, частота в 2D, 3D-режимах и при разгоне)
- сведения о поддержке DirectX
- информацию о видеопамяти (тип, объем, разрядность шины)

<img width="488" height="580" alt="image" src="https://github.com/user-attachments/assets/dce4a32a-8570-4bcc-b5cb-58e8395de52f" />


#### FurMark

FurMark — это утилита для стресс-тестирования видеокарт, которая создает экстремальную нагрузку на GPU с целью проверки стабильности и температурного режима[citation:1][citation:7]. Программа использует OpenGL и симулирует сложные графические эффекты (мех с высоким уровнем детализации). 

#### FluidMark

FluidMark — это аналог FurMark, тестирующий движок физических вычислений **NVIDIA PhysX**[citation:4]. Тест симулирует гидродинамические расчеты (поведение жидкостей) силами видеокарты или процессора.

### Вывод о качестве установленного видеоадаптера

В ходе выполнения лабораторной работы была протестирована видеокарта **NVIDIA GeForce GTX 1660 Ti**. На основе полученных данных можно сделать следующие выводы:

1. **Стабильность работы:** Видеокарта успешно прошла стресс-тестирование в FurMark и FluidMark без выявления артефактов, зависаний или критических ошибок.

2. **Температурный режим:** Максимальная температура GPU составила 76°C в режиме максимальной нагрузки с бубликом. Это значение находится в пределах допустимого диапазона для данной модели (критическая температура около 95°C).

3. **Производительность:** Средний FPS в FurMark (78) и FluidMark (94) свидетельствует о достаточной производительности видеокарты для современных игр в разрешении Full HD на средних и высоких настройках.

4. **Система охлаждения:** Разница в температуре между тестом с бубликом (76°C) и без бублика (72°C) составляет всего 4°C, что говорит об эффективной работе системы охлаждения.

**Общая оценка:** Видеоадаптер находится в хорошем техническом состоянии, стабильно работает под нагрузкой и соответствует своим паспортным характеристикам. Рекомендуется к использованию для игр и ресурсоемких графических приложений.

#### 1. Какие другие программы используют для тестирования видеокарт?

| № | Программа | Назначение |
| :--- | :--- | :--- |
| 1 | **3DMark** | Наиболее известный бенчмарк для оценки игровой производительности видеокарт, ставший де-факто отраслевым стандартом[citation:5]. Включает тесты, имитирующие реальные игры, и синтетические тесты возможностей GPU. |
| 2 | **OCCT Perestroïka** | Комплекс утилит для стресс-тестирования основных компонентов: видеокарты, процессора, блока питания[citation:2][citation:8]. |
| 3 | **AIDA64** | Многофункциональная программа-комбайн для диагностики и тестирования, включающая стресс-тесты GPU, GPGPU (вычисления на видеокарте), диска, памяти и кэша[citation:2][citation:8]. |
| 4 | **GPU Caps Viewer** | Программа просмотра сведений об OpenGL драйвере и тестирования OpenGL-подсистемы[citation:2]. |
| 5 | **CUDA-Z** | Утилита для разработчиков, работающих с API CUDA. Показывает характеристики и производительность вычислений на GPU[citation:2][citation:5]. |
| 6 | **VRMark** | Пакет для оценки готовности ПК к играм с поддержкой виртуальной реальности (HTC Vive, Oculus Rift). Поддерживает 5K-рендеринг[citation:5]. |
| 7 | **DirectCompute Benchmark** | Бенчмарк для проверки производительности API универсальных вычислений DirectCompute (входит в DirectX 11) и OpenCL[citation:2]. |
| 8 | **MSI Kombustor** | Аналог FurMark, также созданный на базе движка Geeks3D. Используется для стресс-тестирования и разгона видеокарт. |
| 9 | **Heaven / Superposition Benchmark (Unigine)** | Популярные бенчмарки для оценки производительности видеокарт в сложных 3D-сценах с использованием передовых графических технологий (Tessellation, Global Illumination). |
| 10 | **Valley Benchmark (Unigine)** | Еще один бенчмарк от Unigine для тестирования видеокарт в живописном горном окружении. |

#### 2. Какие программы используют для тестирования мониторов?

| № | Программа | Назначение |
| :--- | :--- | :--- |
| 1 | **PassMark MonitorTest** | Профессиональная утилита для тестирования мониторов и LCD-экранов. Генерирует 35 различных тестовых паттернов, включая тесты на HDR, битые пиксели, равномерность подсветки, цветопередачу, контрастность и время отклика[citation:6]. |
| 2 | **Display Test App (Web)** | Веб-приложение для оценки качества дисплея. Включает 12 специализированных тестов: обнаружение битых пикселей, проверка равномерности, измерение времени отклика и др. Работает через браузер без установки[citation:3]. |
| 3. | **screentest** | Небольшая утилита для Linux (РЕД ОС) с набором тестовых экранных шаблонов для быстрой оценки качества монитора и выявления дефектов[citation:9]. |
| 4. | **EIZO Monitor Test** | Бесплатная программа от известного производителя мониторов EIZO. Содержит набор тестов для настройки и проверки: равномерность подсветки, цветопередача, углы обзора, сходимость лучей, оценка качества градиентов. |
| 5. | **Nokia Monitor Test** | Классическая портативная утилита (не обновляется с 2000-х годов, но до сих пор популярна). Позволяет проверить геометрию, сходимость, муар, разрешение, контрастность, битые пиксели на LCD и CRT-мониторах. |
| 6. | **Dead Pixel Tester (DPT)** | Простая утилита для поиска битых пикселей. Заполняет экран различными цветами (красный, зеленый, синий, белый, черный), на фоне которых дефектные пиксели становятся заметны. |
| 7. | **UFO Test (Blur Busters)** | Онлайн-тест для оценки времени отклика и инерционности пикселей (черепашки, UFO). Позволяет оценить качество разгона матрицы и наличие шлейфов при движении объектов. |
| 8. | **Lagom LCD Monitor Test** | Набор веб-страниц с тестовыми паттернами для проверки контрастности, гаммы, цветопередачи, времени отклика, равномерности подсветки и острых углов обзора. |
| 9. | **Monitor Calibration Wizard** | Утилита для профессиональной калибровки и профилирования мониторов, позволяющая настроить гамму, цветовую температуру, яркость и контрастность с использованием тестовых образцов. |
